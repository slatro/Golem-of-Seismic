# Seismic Golem Unity Scaffold

This folder is a clean Unity-ready project scaffold for `Seismic Golem`.

## Intent
- keep the exact golem silhouette as the only playable hero
- keep the exact pink faceted chest Core as the character's identity center
- start with a graybox combat prototype before expanding into full run flow

## What Is Included
- `Assets/` folder structure aligned with the technical architecture
- starter C# shell scripts for core gameplay systems
- `Docs/` shortcuts to the design package in the parent folder

## Suggested First Use
1. Create a new Unity URP project.
2. Copy this scaffold into the Unity project's root `Assets` area as needed.
3. Start with `Scenes/Prototype/Prototype_CombatArena.unity`.
4. Wire up `PlayerRoot`, `PlayerMotor`, and `PlayerCombat` first.

## Immediate Priorities
1. Import the exact golem character model.
2. Import the exact pink chest Core model or texture setup.
3. Lock camera angle and player scale.
4. Make one enemy interaction fun before building room flow.

## Design Docs
Core design docs live one level up in the current workspace and are mirrored conceptually into `Docs/`.
