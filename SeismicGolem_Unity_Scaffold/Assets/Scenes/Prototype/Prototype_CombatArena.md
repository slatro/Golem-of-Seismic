# Prototype_CombatArena

Use this scene as the first real implementation target.

## Place In Scene
- one player spawn
- one medium graybox arena
- four enemy spawn markers
- one debug canvas
- one camera rig

## Test Order
1. Tunnel Brute
2. Shard Mite swarm
3. Vein Caster mixed room
4. Obsidian Sentinel

## Exit Rule
Do not build the room-to-room prototype until movement, combo, heavy attack, and Core Pulse already feel good here.
