# Seismic Golem

## Task Breakdown V1

Bu liste production başlangıcı için pratik görev kırılımıdır.

## Milestone 1: Slice Foundations

- `M1-01` Oyunun referans karakter modelini lock et
- `M1-02` Opening 10 minutes dokümanını playable beat'lere böl
- `M1-03` Sanctum alanının top-down layout sketch'ini hazırla
- `M1-04` Uyanış odası için giriş kadrajını belirle
- `M1-05` İlk vista alanının kadrajını belirle

## Milestone 2: World Blockout

- `M2-01` Uyanış odası greybox
- `M2-02` İlk koridor greybox
- `M2-03` İlk altar alanı greybox
- `M2-04` Mühür bahçesi greybox
- `M2-05` Vista platformu greybox
- `M2-06` Bütün akış için traversal test

## Milestone 3: Core Gameplay

- `M3-01` Ağır yürüyüş controller
- `M3-02` Yumruk sistemi
- `M3-03` Core pulse sistemi
- `M3-04` Breakable taş bariyer
- `M3-05` Altar node aktivasyonu
- `M3-06` Kapı açılma sistemi
- `M3-07` Checkpoint sistemi

## Milestone 4: Encounter Layer

- `M4-01` İlk küçük guardian enemy
- `M4-02` Daha dayanıklı ikinci enemy
- `M4-03` Basit combat arena test
- `M4-04` Shard pickup sistemi
- `M4-05` İlk combat ödül anı

## Milestone 5: Environment Identity

- `M5-01` Taş yüzey materyal dili
- `M5-02` Sanctum floor pattern
- `M5-03` Ray line visual language
- `M5-04` Altar form language
- `M5-05` Sütun ve büyük kütle seti
- `M5-06` Arka plan boşluk kompozisyonu

## Milestone 6: Narrative Layer

- `M6-01` İlk 5 yazıtı sahneye yerleştir
- `M6-02` İlk sanctum node adını yerleştir
- `M6-03` Açılışta rayın uyanma anını bağla
- `M6-04` Vista öncesi kısa çevresel hikaye anı

## Milestone 7: Feel Layer

- `M7-01` Kamera idle drift
- `M7-02` Vista kamera açılımı
- `M7-03` Yumruk micro shake
- `M7-04` Adım sesleri
- `M7-05` Yumruk impact sesi
- `M7-06` Core hum katmanı
- `M7-07` Sanctum ambience pass

## Milestone 8: Polish

- `M8-01` UI sadeleştirme
- `M8-02` Gereksiz feedback temizliği
- `M8-03` Işık ayarı
- `M8-04` Fog / haze tuning
- `M8-05` Capture-ready pass

## Öncelik Sırası

İlk yapılması gerekenler:

1. layout
2. movement + yumruk
3. altar + kapı sistemi
4. first encounter
5. vista

## Not
